# Hamza Shoukat — Portfolio

> **Turning Vision into Virtual Reality**

Personal portfolio of **Hamza Shoukat**, Senior AI Python Developer — an advanced, animated, interactive single-page site built with React.

## ✨ Highlights

- **Cinematic dark UI** — glassmorphism, ambient light blobs, grid backdrop, cursor-tracking glow
- **Neural-network particle field** — interactive canvas in the hero that reacts to your pointer
- **Playable terminal** — a working in-page shell (`help`, `skills`, `projects`, plus a few easter eggs — try `sudo hire hamza`)
- **3D tilt project cards** with category filtering and layout animations
- **Magnetic buttons**, typewriter roles, animated count-up stats, timeline with an animated spine
- **Accessible & respectful** — keyboard-friendly, `prefers-reduced-motion` support, semantic markup

## 🛠 Tech Stack

| Layer      | Tools                      |
| ---------- | -------------------------- |
| Framework  | React 19 + Vite            |
| Styling    | Tailwind CSS v4            |
| Animation  | Framer Motion + Canvas API |
| Icons      | Lucide React               |
| Typography | Inter + JetBrains Mono     |

## 🚀 Getting Started

```bash
# install dependencies
npm install

# start the dev server
npm run dev

# production build
npm run build

# preview the production build
npm run preview
```

## 📂 Structure

```
src/
├── components/    # Hero, Skills, Projects, Terminal, …
├── data/          # profile.js — all content in one place
├── App.jsx
├── index.css      # Tailwind v4 theme tokens + custom effects
└── main.jsx
```

All content (bio, skills, projects, experience, testimonials) lives in [`src/data/profile.js`](src/data/profile.js) — edit one file to update the whole site.
